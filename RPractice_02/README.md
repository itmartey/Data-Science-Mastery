# Data Wrangling and Manipulation – RPractice_02 

This repository contains an R Markdown project demonstrating data cleaning, transformation, and exploratory analysis using real-world health datasets.

## Project Overview

**Title**: Data Wrangling and Manipulation  
**Author**: Isaac Tetteh Martey  
**Date**: March 2026  
**Output formats**: HTML

This document showcases common data wrangling tasks using the **tidyverse** ecosystem in R, including:

- Loading and inspecting raw data
- Cleaning column names and data types
- Handling missing values
- Creating derived variables (age groups, BMI, flags, categories)
- Filtering, selecting, mutating, grouping & summarising
- Reordering columns meaningfully
- Visualising distributions (boxplots, bar charts)
- Using modern table formatting (`gt`, `reactable`)

## Datasets Used

Three public datasets from [The Graph Courses](https://www.thegraphcourses.org):

1. **Yaoundé COVID-19 Survey** (~971 observations)  
   - Source: https://raw.githubusercontent.com/the-graph-courses/idar_book/refs/heads/main/data/yaounde_data2.csv

2. **Sarcopenia in Elderly** (~239 observations)  
   - Source: https://raw.githubusercontent.com/the-graph-courses/idar_book/refs/heads/main/data/sarcopenia_elderly.csv

2. **Flu incidence in China** (~136 observations)  
     - Source: https://raw.githubusercontent.com/the-graph-courses/idar_book/refs/heads/main/data/flu_h7n9_china_2013.csv

## Main Sections Covered

- Data ingestion & initial exploration
- Column selection & reordering (bringing outcomes first)
- Creating binary flags (no symptoms, low grip strength, pregnancy status)
- Derived variables (BMI, age groups, pregnancy classification)
- Subsetting by categories (neighborhood, sex, etc.)
- Handling & summarizing missing values
- Basic visualisation (boxplots of grip strength by sex)
